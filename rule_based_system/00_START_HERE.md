### Example 3: Process File
```python
from rule_based_system import run_pipeline

```

### Example 4: Export to Excel
```python
from rule_based_system.exports.export_handler import export_excel

export_excel(test_cases, "my_tests.xlsx")
```

### Example 5: REST API
```bash
# Start server
python3 main.py

# Upload file (in another terminal)
curl -X POST http://localhost:8001/generate \
  -F "file=@requirements.pdf"

# Or send text
curl -X POST http://localhost:8001/generate/text \
  -H "Content-Type: application/json" \
  -d '{"text":"Your requirement here"}'
```

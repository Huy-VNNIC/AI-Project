\subsection{Phương án 2: Dùng PDF Annotation (Khuyến nghị)}

\begin{enumerate}
    \item Compile \texttt{Paper\_v2/main.tex} thành PDF bình thường (không sửa gì)
    \item Mở PDF bằng Adobe Acrobat Reader / Foxit Reader / Xournal++
    \item Dùng công cụ ``Highlight Text'' để bôi vàng theo bảng trên
    \item Thêm comment annotation: ``Added per Reviewer X, Comment Y''
    \item Export thành PDF có annotations
\end{enumerate}

\subsection{Phương án 3: Tạo bản diff màu sắc}

Dùng package \texttt{latexdiff} để tạo PDF so sánh:

\begin{verbatim}
latexdiff old_version.tex Paper_v2/main.tex > diff.tex
pdflatex diff.tex
\end{verbatim}

Phần thêm mới sẽ tự động được đánh dấu xanh (addition) hoặc đỏ (deletion).

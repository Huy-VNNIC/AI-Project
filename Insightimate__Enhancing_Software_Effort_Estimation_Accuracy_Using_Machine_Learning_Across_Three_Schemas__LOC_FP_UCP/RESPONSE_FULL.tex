\section*{Executive Summary of Major Revisions}

\begin{enumerate}[leftmargin=*]
\item \textbf{Dataset Expansion (+192\%):} n=1,042 $\rightarrow$ n=3,054 projects across 18 sources (1979-2023); FP schema: n=24 $\rightarrow$ n=158 (+558\%)
\item \textbf{State-of-the-Art Models:} Added XGBoost (MAE 13.24 PM vs RF 12.66 PM, <5\% difference)
\item \textbf{Enhanced Metrics:} Added MdMRE and MAPE for robust statistics
\item \textbf{Cross-Source Validation:} LOSO validation on 11 LOC sources (21\% MAE degradation--acceptable)
\item \textbf{Calibrated Baseline:} Training-data-fitted power-law baseline ($E = A \times \text{Size}^B$)
\item \textbf{Methodological Transparency:} Macro-averaging protocol, dataset provenance, deduplication rules, bootstrap CI
\item \textbf{Expanded Literature:} 7 new papers cited (3 IEEE journals, 2 preprints)
\item \textbf{Improved Presentation:} 300 DPI figures, proper numbering, line numbers, linguistic revision
\end{enumerate}

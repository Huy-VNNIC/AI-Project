## 🚀 HOW TO USE THE SYSTEM

### 1. Basic Usage
```python
from ai_test_generation_v2_enhanced import EnhancedTestCaseGeneratorV2

generator = EnhancedTestCaseGeneratorV2()
requirements = ["User login with password", "Search functionality", "Payment processing"]

tests, metrics = generator.generate_comprehensive_tests(requirements)

for test in tests:
    print(f"{test.test_id}: {test.description}")
    print(f"  Category: {test.category.value}")
    print(f"  Confidence: {test.confidence:.2%}")
```

### 2. Threat Analysis
```python
from threat_modeling_engine import ThreatModelingEngine

engine = ThreatModelingEngine()
requirement = "System allows user search with HTML display"

threats = engine.identify_threats_in_requirement(requirement)
scenarios = engine.generate_attack_scenarios(requirement)

for scenario in scenarios:
    print(f"Threat: {scenario}")
```

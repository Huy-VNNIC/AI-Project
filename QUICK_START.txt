Change Primary Color (Edit any HTML file, inside <style>):

   :root {
       --primary: #006666;    /* Change this hex color */
   }

   html[data-theme="dark"] {
       --primary: #00D9D9;    /* Dark mode primary */
   }

Popular Color Presets:
   • Blue:   --primary: #0066FF;  (Professional)
   • Green:  --primary: #059669;  (Enterprise)
   • Purple: --primary: #7C3AED;  (Modern)

Problem: "Command not found" when running start script
   Solution: chmod +x start_neumorphism_ui.sh

Problem: Port 8000 already in use
   Solution: 
      • Kill existing: lsof -i :8000 | awk 'NR>1 {print $2}' | xargs kill
      • Or use different port: export API_PORT=8001

Problem: Page shows 404
   Solution: Check files exist: ls -la templates/*neumorphism*

Problem: Dark mode not working
   Solution: Open DevTools (F12) > Console and check for JavaScript errors

Problem: Fonts not loading
   Solution: Check Network tab in DevTools, ensure Google Fonts loads
